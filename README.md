# Overview

Welcome to the official **SurveySentinel Examples** repository.

This repository provides publicly available demonstration material to help users understand how SurveySentinel integrates into professional aerial survey workflows.

The examples contained here are intended for learning, evaluation and demonstration purposes only.

---

# Repository Purpose

This repository will contain a growing collection of practical examples including:

- Example survey projects
- Sample datasets
- Flight planning examples
- Workflow demonstrations
- Configuration examples
- Integration examples
- Screenshots
- Tutorials
- Reference files

No proprietary source code or internal engineering documentation is contained within this repository.
